#  California House Price Prediction

A Machine Learning web application that predicts California house prices based on different housing features using FastAPI, Machine Learning, HTML, and CSS.

---

#  Project Overview

The California House Price Prediction project is an end-to-end Machine Learning application designed to estimate house prices using housing-related input features.

This project combines:
- Machine Learning
- FastAPI backend
- HTML/CSS frontend
- Model deployment concepts

Users can enter housing details through a web interface and receive predicted house prices instantly.
